## 🤔 O que é o Prever?

O **Prever** é um jogo social de prognósticos onde você aposta pontos virtuais em eventos do mundo real — política, economia, esportes, entretenimento e muito mais.

Quanto mais certeiro você for, mais pontos acumula. Suba no ranking e prove que você sabe ler o futuro melhor que todo mundo.

> ⚠️ O Prever usa **pontos virtuais** — não há dinheiro real envolvido.

---

## 🚀 Como começar

1. Abra o Telegram e acesse [@Prever_bot](https://t.me/Prever_bot)

4. Veja os mercados abertos com `/mercados`

## 🎲 Como apostar

Quando você digitar `/mercados`, o bot vai mostrar cada mercado com dois botões:

```
✅ SIM — 35¢     ❌ NÃO — 65¢
```

**Opção 1 — Botões:**
Toque em **SIM** ou **NÃO** e depois informe o valor que quer apostar.

**Opção 2 — Resposta direta:**
Responda a mensagem do mercado com o valor e o lado:
```
10 sim
50 nao
```

---

## 💰 Entendendo os preços (odds)

Cada mercado tem dois preços — **SIM** e **NÃO** — que somam sempre 100¢.

| Preço SIM | Significado |
|---|---|
| **20¢** | Evento improvável — se acertar, ganho alto |
| **50¢** | Evento incerto — chances iguais |
| **80¢** | Evento provável — se acertar, ganho menor |

**Como calcular o ganho:**
```
Ganho = Valor apostado ÷ Odd
```

**Exemplo:**
- Você aposta **100 pontos** em SIM a **35¢**
- Se SIM ganhar → você recebe **285 pontos** (lucro de 185)
- Se NÃO ganhar → você perde os 100 pontos apostados

## ⚖️ Regras

1. **Sem dinheiro real** — apenas pontos virtuais
2. **Uma aposta por mercado por usuário** — sem manipulação
3. **Mercados são resolvidos pelo administrador** com base em fatos verificáveis
4. **Resultados são definitivos** — não há recurso após resolução
5. **Saldo zerado?** Entre em contato com o admin para reposição de cortesia
